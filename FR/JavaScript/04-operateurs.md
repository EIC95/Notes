# Opérateurs en JavaScript

## Opérateurs Arithmétiques
| Opérateur | Exemple  | Signification     |
|-----------|----------|-------------------|
| `=`       | `x = y`  | `x` devient `y`   |
| `+=`      | `x += y` | `x` = `x + y`     |
| `-=`      | `x -= y` | `x` = `x - y`     |
| `*=`      | `x *= y` | `x` = `x * y`     |
| `/=`      | `x /= y` | `x` = `x / y`     |
| `%=`      | `x %= y` | `x` = `x % y`     |
| `**=`     | `x **= y`| `x` = `x ** y`    |

## Opérateurs de Comparaison
| Opérateur | Description en français            |
|-----------|------------------------------------|
| `==`      | égal à                             |
| `===`     | même valeur et même type            |
| `!=`      | différent de                        |
| `!==`     | valeur différente ou type différent |
| `>`       | supérieur à                         |
| `<`       | inférieur à                         |
| `>=`      | supérieur ou égal à                 |
| `<=`      | inférieur ou égal à                 |
| `?`       | opérateur ternaire                  |

## Opérateurs Logiques
| Opérateur | Description           |
|-----------|-----------------------|
| `&&`      | et logique            |
| `\|\|`      | ou logique            |
| `!`       | non logique           |

## Autres Opérateurs
| Opérateur     | Description                              |
|---------------|------------------------------------------|
| `typeof`      | Retourne le type d'une variable          |
| `instanceof`  | Retourne vrai si un objet est une instance d'un type d'objet  |
